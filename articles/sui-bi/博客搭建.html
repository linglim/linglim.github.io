Blog搭建

Date 2016-09-11(星期日) 00:00 By Linglim Category 随笔. Tags 博客

我的博客搭建：

keys :pelican+github enviroment:win10 + python

由于是windows环境搭建出现很多困难，需准备很多python 的环境

introduction:

  • pelican:python写的的静态blog生成器，支持markdown，有许多主题和插件，易配置
  • setuptools：python 包管理工具，功能：利用EasyInstall自动查找、下载、安装、升级依赖包

    Pelican is a static site generator, written in Python. Highlights include:markdown formats、simple CLI tool、Easy to interface、Completely static output

install preparation ：

  1. python :Python 3.4.4

  2. setuptools :win复制脚本，双击运行（http://blog.csdn.net/robertkun/article/details/9751123）

  3. pip 安装(参考) 参考1

安装pelican

`pip install pelican`

创建博客站点

mkdir blog //创建文件夹，名称可根据自己喜欢定
cd blog
pelican-quickstart

blog目录

blog/

├── content # 存放输入的markdown或RST源文件 
│ └── (pages) # 存放手工创建的静态页面，可选 
│ └── (articles) # 存放手工创建的文章，可选 
├── output # 存放最终生成的静态博客 
├── develop_server.sh # 测试服务器 
├── Makefile # 管理博客的Makefile 
├── pelicanconf.py # 配置文件 
└── publishconf.py # 发布文件，可删除

配置文件pelican.py

主题下载：git clone https://github.com/getpelican/pelican-themes.git 插件下载：git clone https://github.com/getpelican/pelican-plugins.git

配置pelican参考

pelican参考

主题:THEME = 'pelican-themes/xx' 插件：PLUGIN = 'pelican-themes/xx'

写博文

markdown 格式，在content页
标签：
Date: 时间
Title: 标题
Category:目录
Tags: 标签
Slug: 博客搭建
URL：指定生成的位置（pages/about.html）
save_as: 保存的位置（pages/about.html）
Status:drafts(草稿：不被看到)

标签解读

-Slug:

  1. 文档唯一标识，html名，必须唯一.pelican.py中有关url设置 ARTICLE_URL = 'articles/{category}/{slug}.html'
  2. 官网介绍是：可两篇文章公用一个slug标签，尽管发布的内容不同（语言不通），仍有唯一的相同点——slug（在此处用作识别符）。如果你并未明确定义slug，必须要确保不同语言版本文章的标题是相同的，因为slug将会根据文章标题自动生成。

  3. pelican.py中有关url设置，

    ARTICLE_SAVE_AS = 'articles/{category}/{slug}.html 说明了生成的html存储位置

Status:草稿，这篇文章也会生成html，但不会放在博客的主页及分类索引里，这样普通的读者一般看不到这个文章。

本地运行预览

在blog目录下，博文在content目录下，生成在output目录， -s:配置文件位置 D:\blog>pelican D:/blog/blog/content -o D:/blog/blog/output -s D:/blog/blog/pelicanconf.py

Done: Processed 1 article, 0 drafts, 0 pages and 0 hidden pages in 2.70 seconds.

预览生成的站点

For Python 2, run:

cd output
python -m SimpleHTTPServer

or

make html

For Python 3, run:

cd output
python -m http.server

or

make serve

在浏览器中输入http://localhost:8000即可看到博文效果。
