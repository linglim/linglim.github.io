中药材项目部署

Date 2016-09-20(星期二) 00:00 By Linglim Category 后端. Tags javaweb

Summary：部署

java web部署

概要

  1. 下载配置服务器环境或直接将你电脑的jdk，tomcat，mysql复制粘贴到服务器

  2. 打成war就可以，拷贝到你的服务器的tomcat下的webapps中，启动服务器，访问：http://IP:port/项目名称

  3. 登陆服务器

详细：

  1. 将工程打包（.war或者.ear）。eclipse下右击项目文件，选择弹出菜单中的“Export”菜单，选择“J2EE”下面的随便一个，点击下一步，选在保存目录，输入文件名后，点击完成就行。

  2. 部署，通过winscp连接远程服务器。把工程文件放入到tomcat安装目录下的webapps下，举例：tomcat6.0\apache-tomcat-6.0.32\webapps。

    连接远程:

    • ip:123.57.72.171（好像是阿里云）

    • windows：可远程桌面登陆

    • linux：putty登陆或云账号登陆

  3. 然后启动tomcat.在tomcat的安装目录下，双击bin/startup.bat

  4. 浏览器输入服务器地址。
