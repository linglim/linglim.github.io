zhongyaocai项目部署

Date 2016-09-20(星期二) 00:00 By Linglim Category 后端. Tags javaweb Lang zh

Summary：部署

java web部署

1、打成war就可以，拷贝到你的服务器的tomcat下的webapps中，启动服务器，访问：http://IP:port/项目名称 2、登陆服务器

关于打包： 1、1.将工程打包（.war或者.ear）。右击项目文件，选择弹出菜单中的“Export”菜单，选择“J2EE”下面的随便那个，点击下一步，选在保存目录，输入文件名后，点击完成就行。 2. 在刚才选择的目录里面找到生成的工程文件，把工程文件放入到tomcat安装目录下的webapps下，就像这个：D:\tomcat6.0\apache-tomcat-6.0.32\webapps。 3. 然后启动tomcat,在tomcat的安装目录下，找到bin目录下的startup文件，点击就可以启动tomcat了。 4. 接下来，在浏览器里面输入url就可以了访问我们想访问的东西了，大功告成。

参考： http://blog.csdn.net/change_on/article/details/49205901
