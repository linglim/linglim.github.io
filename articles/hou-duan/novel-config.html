控制器Controller 负责处理由DispatcherServlet 分发的请求，它把用户请求的数据经过业务处理层处理之后封装成一个Model ，然后再把该Model 返回给对应的View 进行展示。

使用： 只需使用@Controller 标记一个类是Controller ，然后使用@RequestMapping 和@RequestParam 等一些注解用以定义URL 请求和Controller 方法之间的映射，这样的Controller 就能被外界访问到。 例如：

@Controller

@RequestMapping ( “/test” )//可不写 public class MyController { @RequestMapping ( “/showView” ) public ModelAndView showView() { ModelAndView modelAndView = new ModelAndView(); modelAndView.setViewName( “viewName” ); modelAndView.addObject( ” 需要放到 model 中的属性名称 ” , ” 对应的属性值，它是一个对象 ” ); return modelAndView; }

} @Controller @RequestMapping ( “/test” )//controller后面加了mapping表示一下的都是在”/test”目录下
